Portfolio

My first 2 major rails projects were Appalachian Healthcare and PizzaOne. Which both of these I built 100% from the ground up and still work on for rainy days. There is always features to be added to projects! Appalachian Healthcare was an Iron Yard project that I decided to restyle, and still implementing features when I have free time. In PizzaOne, I was able to implement Dropbox API to upload images in Dropbox. CoDependent was a fun challenge with some Front-End Engineers that we built in just 2 days. In CoDependent I built the whole back end for the Front End Engineers to display their information.
